# Electronic Journal System

Электронная система журнала для преподавателей и студентов.
Проект разрабатывается как полноценная web-система с frontend и backend архитектурой.

---

# Tech Stack

## Frontend

- React
- Vite
- TailwindCSS
- JavaScript
- React Hooks

## Backend

- Node.js
- Express.js
- PostgreSQL
- Prisma ORM
- JWT Authentication
